# progress-report-tt

A bilingual (EN/JA) learning-progress dashboard, published via GitHub Pages.

## Overview

`index.html` is a self-contained Kanban-style snapshot of takumi's learning
progress (WarmUps drill sections, explainer articles, English study). It is
auto-generated by the `progress-report-tt` Claude Code skill from conversation
logs and live WarmUps app data — do not edit it by hand.

## Usage

Open the published page and use the EN / JA toggle (top right) to switch
languages, the theme toggle for light/dark, and click any card for details.

## Configuration

Regeneration and publishing are driven by the skill at
`~/.claude/skills/progress-report-tt/` on the author's machine. The page sets
`noindex` and this repo ships a `robots.txt` so the report stays out of search
engines; access is by direct URL.
